# Duline – Flutter Frontend

Mobile application built with Flutter for the Duline family task management system.

## Features
- User authentication
- Family groups
- Shared tasks
- Real-time task updates
- Responsive Flutter UI

## Tech Stack
- Flutter (Dart)
- REST API integration
- Spring Boot backend

## Run the project
flutter pub get
flutter run
